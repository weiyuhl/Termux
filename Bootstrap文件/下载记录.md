## 🎯 用途说明

### 在编译中的作用

1. **编译时嵌入**
   - 通过汇编 `.incbin` 指令嵌入到 `libtermux-bootstrap.so`
   - 成为 APK 的一部分

2. **运行时解压**
   - 用户首次启动 Termux 时
   - 从 `.so` 文件读取 ZIP 数据
   - 解压到 `/data/data/com.termux/files/usr/`

3. **提供 Linux 环境**
   - 包含完整的 Linux 用户空间工具
   - 使 Termux 能够运行 Linux 程序

## 📊 架构选择

### 如何选择合适的架构？

```
查看设备架构:
$ adb shell getprop ro.product.cpu.abi

常见结果:
- arm64-v8a    → 使用 bootstrap-aarch64.zip
- armeabi-v7a  → 使用 bootstrap-arm.zip
- x86_64       → 使用 bootstrap-x86_64.zip
- x86          → 使用 bootstrap-i686.zip
```

```
现代手机 (2015+):
- 95% ARM64 (aarch64)
- 5% ARM32 (arm)

模拟器:
- 80% x86_64

```
